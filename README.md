# @ramses-superapp/mdk

Mini Development Kit — contributor CLI for the Ramses Super App platform.

## Install

```bash
npm install -g @ramses-superapp/mdk
```

## Quick start

```bash
mdk bootstrap          # one-shot: clone SDK + Engine, install, verify
mdk feature start <name> --type api --family <family>
# ...edit SSOT + handler...
mdk feature submit
```
